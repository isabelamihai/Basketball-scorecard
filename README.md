# Basketball-scorecard
The app is a game scorecard that adds points for 2 teams using 3 buttons linked to 3 functions.

## Tehnologii folosite

- HTML
- CSS
- JavaScript

## Cum se folosește

1. Clonează repository-ul
2. Deschide index.html în browser
